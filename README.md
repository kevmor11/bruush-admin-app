## Knex Commands

 - Once migrations are written - `knex migrate:latest` to run the migration
 - `knex migrate:rollback` to rollback the most recent batch of migrations

## .env Contents

 - SHOPIFY_API_KEY
 - SHOPIFY_API_PASSWORD
 - DATABASE_HOST
 - DATABASE_USER
 - DATABASE_PASSWORD
 - DATABASE_NAME
